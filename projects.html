Weather App

This project is a simple weather application that allows users to check the current weather conditions of any city. It uses a clean user interface and fetches real-time weather data using a weather API. The app displays information such as temperature, humidity, and weather conditions (like sunny or rainy). It was built using basic HTML, CSS, and JavaScript, and it helped me understand how APIs work in real-world applications.
